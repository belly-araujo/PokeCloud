Tecnologias utilizadas
  1. Node.js
  2. Exprees
  3. Mysql2
  4. React

Como rodar o projeto
  1. Clona o repositório
    git clone https://github.com/belly-araujo/PokeCloud.git
  2. Entrar na pasta do projeto
       cd PokeCloud
  3. Entrar na pasta do backend
       cd backend
  4. Instalar dependências do back
       npm install
  5. Rodar o servidor do back
      npm start
  6. Abra outro terminal e entre na pasta frontend
      cd frontend
  7. Instalar dependências do front
      npm install
  8. Rodar o projeto no front
      npm start

Para o banco de dados, importe o arquivo .sql e rode na sua máquina.
